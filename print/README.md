# print/ — canal de imagens pro Claude

Pasta temporária pra trocar imagens com o assistente (Claude Code).

**Fluxo:**
1. Você faz upload das imagens aqui (PNG/JPG) pela web do GitHub.
2. O assistente dá `git pull`, abre e analisa as imagens.
3. Depois de usadas, o assistente apaga as imagens (commit de limpeza).

Nada aqui é parte do plugin — a pasta não vai pro build (`output/`).
